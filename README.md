## 🚀 Instalação

```bash
# Clone o repositório
git clone https://github.com/DUCAT1-blip/Cart-o-De-Visista.git

# Entre no diretório do repositório
cd Cart-o-De-Visista

# Instale as dependências
npm install
# ou
yarn install
```
